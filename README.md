This repository is for managing an Air Quality Sensor project for my Electronics class. 

I have no idea what I'm doing tbh. I'll deal with the consequences later (Third restructuring of this repository gotta love OCD).

## Functionality
The Air Quality Sensor will be made using an ESP32 S3 devkit module and will consist of three sensors:
+ SHT31 a Humidity & Temperature Sensor
+ SenseairS8, a Co2 Sensor
+ PMS7003, a Particulate Matter Sensor

## Navigating the Repository
### Code
All the code that I will be making throughout the project.
+ InDev/ - All the in-progress arduino sketches.
+ main/ - The main script that I will be plugging into the Air Quality Sensor.
+ others/ - Example code from tutorials.

### Assets
